
# NestJS 

## Setting up Development Environment 

Install the latest version of NodeJS

### using npm 
npm install -g @nestjs/cli

### Using Yarn
yarn global add @nestjs/cli

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
